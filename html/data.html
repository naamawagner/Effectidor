Effectidor identifies T3SS and flagella proteins in the input genome based on an internal datasets of the different T3SS types. These proteins are excluded from the analysis of Effectidor and are reported separately.
These datasets are available for downloading:
Chlamydiales
Ysc
SPI-1
SPI-2
Hrp1
Hrp2
Rhc
Flagella

Effectidor uses a type-III effectors dataset for homology search as well as for composing the positive group if no effectors file is supplied.
